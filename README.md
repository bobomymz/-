# 🌍 AI 地形生成与战争模拟器 (v1.0)

欢迎来到这个基于浏览器的小型“造物主”实验室。本项目是一个高度集成的程序化生成（Procedural Generation）展示工具，结合了**柏林噪声地形渲染**、**A* 路径规划水系算法**以及一套复杂的**自驱动 AI 战争模拟系统**。

---

## 🚀 核心功能

* **程序化地形生成**：利用多层柏林噪声（Perlin Noise）生成包含平原、山脉、雪顶的自然景观。
* **动态水系模拟**：基于 A* 算法模拟河流从高山流向低洼处或大海的物理路径。
* **AI 势力博弈**：多个阵营（Faction）自动扩张、占领领土、建立补给线并进行交战。
* **复杂单位行为**：模拟单位包含 **健康度 (Health)**、**士气 (Morale)**、**补给 (Supply)** 以及独特的 **背叛 (Betrayal)** 机制。
* **实时数据统计**：内置单位数量与占领格数双榜单，实时反映战场态势。

---

## 🛠 技术深度解析

### 1. 地形与降噪 (Terrain & Noise)
地形生成不仅是简单的随机数，而是使用了多层频率叠加的 **分形布朗运动 (FBM)**。
* **基础起伏**：使用标准 Perlin 噪声。
* **山脊细节**：通过对噪声取绝对值并反转，产生尖锐的山脉褶皱。
* **高度映射**：应用幂函数 $h = Math.pow(h, 1.2 - h^2)$，让低海拔地区更加平缓，高海拔地区更加陡峭。

### 2. A* 侵蚀河流算法
不同于简单的随机漫步，本系统的河流生成极具逻辑性：
* **源头提取**：算法会自动寻找地图中的高海拔“山峰”作为水源。
* **路径寻优**：河流利用 A* 算法寻找向下的“代价最小路径”。其代价函数定义为：
    $$Cost = Height^{4} + Random\_Noise + River\_Penalty$$
* **侵蚀效果**：河流经过之处会动态降低地形高度，形成河谷，并具有动态宽度（下游宽，险峻处窄）。

### 3. 战争生态系统 (War Ecosystem)
这套模拟系统的精髓在于其“涌现行为”：
* **领土涂色**：单位根据自身实力和地形坡度影响周围单元格的归属权。
* **补给线系统**：远离己方定居点的单位会损耗补给，最终导致健康度下降。
* **黑天鹅背叛机制**：当单位士气崩溃时，可能发生背叛。更精妙的是，代码中设计了“阵营倾向性叛变”，一旦触发，可能导致整个战线像多米诺骨牌一样连锁崩溃。
* **波传导机制 (Propagate Waves)**：模拟了信息的传递。前线的战斗会通过“波”的形式召唤支援，同时影响后方士兵的士气，模拟了战役感。

## 🎮 如何使用

1.  **生成世界**：点击 `重新生成地图`。程序会重新计算高度图、分布定居点并规划河流。
2.  **启动战争**：点击 `开始战争模拟`。各阵营将从各自的首都开始自动产兵并向外扩张。
3.  **战术观察**：
    * 点击 `显示单位位置` 查看具体的兵力分布（圆点）。
    * 观察颜色覆盖范围：深色表示领土控制力强，浅色表示控制力薄弱。
    * **黄金定居点**：位于河岸的定居点（金边标记）具有渡口属性，能极大降低跨河移动的代价。

## 📜 开发者说明

该项目由单文件 HTML 实现，依赖原生 JavaScript 的 `requestAnimationFrame` 进行高效渲染。
* **性能优化**：地形渲染使用了像素级 Canvas 操作。
* **扩展建议**：你可以通过修改 `SIZE` 常量来调整地图大小，或调整 `STRENGTH_DECAY` 来改变战争的节奏。
